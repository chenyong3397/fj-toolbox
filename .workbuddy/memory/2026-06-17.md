# 2026-06-17 工作日志

## 福建省建设行业信息公开平台 - 数据抓取

**最终方案**：通过逆向工程确定正确API端点为 `/credit/tendering/forewarning/qualify-check`，该端点返回省内企业-质量检测的资质异常数据。

**关键发现**：
- `tendering-extract` API返回的是施工企业资质异常（2556条），不是质量检测企业
- 正确API: `/credit/tendering/forewarning/qualify-check`，返回质量检测企业资质预警数据
- 证书类型字段 `certificationTypeName` = "质量检测资质证书（新）"
- 数据字段: companyName, socialCreditCode, qualifyName, qualifyNumber, certificationTypeName, reason, checkTime, safetyCheck

**结果**：
- 异常企业: 31家
- 异常记录: 84条
- 输出文件: `省内企业_质量检测_资质异常.xlsx`

**技术要点**：
- pageSize最小值=10, 最大值=30
- 滑块验证码通过alpha通道模板匹配破解
- 验证码流程: GET /captcha → POST /captcha?captchaType=N&code=X → 后续API带captcha-token头
